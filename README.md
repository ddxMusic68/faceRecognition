# Face Recognition

This Python script is designed to perform face recognition using OpenCV and Haar Cascade algorithm. It can detect and recognize faces in both images and videos.

## Versions

numpy==2.4.2
opencv-contrib-python==4.13.0.90
opencv-python==4.13.0.90

## Installation

```
pip clone "https://github.com/ddxMusic68/faceRecognition"
pip install -r requirements.txt
```
